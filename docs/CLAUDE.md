# 弾丸ソウル 2026.06.17 — 夫婦専用旅行ガイドWebアプリ

## プロジェクト概要

北邑靖晃（きたむら やすあき）・ひとみ夫婦による2026年6月17日（水）の韓国日帰り旅行ガイドアプリ。
スマホでの閲覧に最適化し、当日の行動指針・会話フレーズ・決済戦略・緊急時対応をすべて1つのWebアプリに集約する。

## アクセス制御

- Google OAuth 2.0でログイン
- **primecool.comドメインのGoogleアカウントのみ許可**（靖晃: yasuaki@primecool.com、ひとみ: hitomi@primecool.com）
- それ以外のGoogleアカウントはログイン画面でブロック

## 技術スタック

- **フロントエンド**: Next.js (App Router) + Tailwind CSS
- **ホスティング**: AWS（S3 + CloudFront、またはAmplify）
- **認証**: NextAuth.js + Google Provider（ドメイン制限）
- **リポジトリ**: GitHub（プライベート）
- **ドメイン**: Value Domainで管理（サブドメインをCloudFrontに向ける）
- **データ**: 静的JSON（trip-data.json）、DBは不要

## デザイン方針

- **モバイルファースト**（iPhone 15 Pro / iPhone SE3 で最適表示）
- 韓国語フレーズは**大きく表示**し、店員に画面を見せて使える
- フォント: Noto Sans KR（韓国語）+ Noto Sans JP（日本語）+ Noto Serif JP（見出し）
- カラー: 暖色系（#c8372d をアクセント、#faf6f0 を背景）
- korean_food_phrases.html のデザイントーンを踏襲する

## ページ構成

### 1. トップページ `/`
- 旅行タイトル・日付・天気（当日）
- 各ページへのナビゲーション（大きなカードボタン）
- 現在時刻に基づいて「今やるべきこと」をハイライト

### 2. タイムライン `/timeline`
- 靖晃・ひとみの並行タイムラインを表示
- 【夫婦】【靖晃】【ひとみ】をカラーで区別
- 各予定をタップすると詳細（備考）が展開
- 現在時刻のインジケーター付き
- データ: trip-data.json の `itinerary` セクション

### 3. 会話フレーズ `/phrases`
- カテゴリ別タブ: 量の調整 / 注文 / 会計 / トラブル / 店別 / 基本
- 韓国語（大）+ カタカナ読み（赤）+ 日本語意味（グレー）
- 「店員に見せる」ボタンで韓国語のみ全画面表示
- データ: trip-data.json の `phrases` セクション
- 既存の korean_food_phrases.html の内容を移植

### 4. 店舗ガイド `/stores`
- 4店舗（瑞源・광천식당・부촌육회・용호동낙지）の詳細
- 住所・営業時間・注文セリフ・決済方法・注意事項
- Google Maps リンク
- データ: trip-data.json の `stores` セクション

### 5. 決済ポートフォリオ `/payment`
- T-money / Mastercard / 現金 の三本柱
- 場面ごとの使い分け表
- WOWPASSは現金化一択の方針
- 現金残高トラッカー（手動入力で残高管理）

### 6. 持ち物・準備チェックリスト `/checklist`
- 出発前に確認する項目（e-Arrival Card、両替、T-moneyチャージ等）
- チェック済みの状態はlocalStorageで保持

### 7. 緊急対応 `/emergency`
- MM808は最終便（22:35発、後がない）
- 脱出プラン: 翌朝の成田行きLCC（13社就航）
- セントレア→ナガスパ案（ギャグ枠だが合理的）
- タクシーでの空港直行（60-80分、6-8万W）
- 鉄道不通時の対応フロー

## データファイル

- `trip-data.json`: 全旅程データ（この会話で確定した内容すべて）
- `korean_food_phrases.html`: 元のHTMLフレーズ集（デザインリファレンス）

### 決済戦略（三本柱）
- **T-money**: 交通（地下鉄・バス）。靖晃15,000W / ひとみ12,000W チャージ済
- **Mastercard**: カード可の店・Global ATMキャッシング
- **現金**: ローカル店・屋台・広蔵市場。両替済み80,000W（大黒屋）
- **WOWPASS**: 買い物に使わない。**夜ソウル駅で現金化一択**（97,100W→96,000W回収）

### 現金戦略
- 朝食（瑞源）: 手持ち現金で（2人4-5万W）
- 大田キャッシング: Global ATM（ウリ/ハナ銀行）で5-10万W補充
- ひとみへの10,000W: **渡さない**（WOWPASS/T-money/カードで自立）
- 大田のWOWPASSキオスク: **期待薄**（ソウル中心の設置）

### イミグレ
- e-Arrival Card: 出発前にオンライン申請**必須**（2026年から紙廃止）
- K-ETA: 2026年12月末まで**免除**
- SES登録: 初回は有人ゲート。審査時に同時登録を打診（早朝は登録センター閉の可能性）
- 早朝04:40着: 空いている＋SESリピーターのバイパス効果で有人ゲートも緩和

### 帰路の安全網（四重構造）
1. AREX直通 19:36（チケット通り）
2. AREX 19:06に無料前倒し（安全網・未使用）
3. AREX一般列車（5-12分毎、T-money）

### 鉄道信頼性
- ソウル地下鉄: ホームドア完備で人身事故リスク低い、直通運転少なく連鎖遅延しにくい
- AREX: 専用線で地下鉄障害から独立、障害実績は数年に1回レベル
- 結論: 東京→羽田より構造的に安全

### 量の調整戦略（4食を10時間に詰め込む）
- 瑞源: 粥は消化が早いのでOK
- 광천식당: 면사리（追加麺）を省いて胃袋温存
- 부촌육회: 2品シェア・味見モード（ユッケ+活きダコ、ご飯なし）
- ナッコプセ: 夕食メイン、締めは腹具合で判断
